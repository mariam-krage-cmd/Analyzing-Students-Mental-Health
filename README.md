# Analyzing-Students-Mental-Health

This project explores and analyzes international student data to investigate how the length of stay (`stay`) impacts the average mental health diagnostic scores. The study focuses on three key diagnostic tests:

- PHQ-9 (`todep`) → Depression screening
- SCS (`tosc`) → Social connectedness scale
- ASISS (`toas`) → Acculturative stress scale

The analysis produces a summary table showing the relationship between length of stay and average scores across these tests.
